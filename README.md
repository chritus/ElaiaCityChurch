# Elaia City Church — Payslip Generator

Salary advice / payslip generator for **Elaia City Church**.

## Features

- Employee information capture
- Earnings: Basic Salary, Mission Travel Allowance, Fuel Allowance
- Deductions: SSNIT, Tax
- Print / Save as PDF using the ECC letterhead

## Local use

Open `index.html` in a browser, fill the form, then click **Generate & Print / Save as PDF**.

## Deploy

This site is published via GitHub Pages from the `main` branch root.
